# Windsurf 换号

一个简洁的 Windsurf 账号快速切换插件。

## 功能

- **当前账号显示** - 显示当前使用的账号
- **切换设置** - 切换后是否刷新窗口
- **添加账号** - 手动添加 Windsurf 账号凭据
- **账号列表** - 管理所有已保存的账号，支持切换/删除

## 使用方法

### 添加账号

1. 点击侧边栏的 "Windsurf 换号" 图标
2. 点击 "添加账号" 按钮
3. 填写以下信息：
   - **邮箱**: Windsurf 账号邮箱
   - **API Key**: 账号的 API 密钥
   - **API Server URL**: API 服务器地址

### 切换账号

1. 在账号列表中找到目标账号
2. 点击 "切换" 按钮
3. 等待切换完成（如果开启了"切换后刷新窗口"，会自动重启）

### 删除账号

1. 点击账号旁边的 🗑 按钮
2. 确认删除

## 安装

```bash
# 进入项目目录
cd windsurf-switch

# 安装依赖
npm install

# 构建
npm run build
```

## 开发

```bash
# 监听模式
npm run watch
```

## 注意事项

- 首次使用时，插件需要对 Windsurf 进行补丁，会自动重启一次
- 账号凭据保存在本地，请妥善保管
